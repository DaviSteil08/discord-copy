# 🚀 Discord-Inspired Premium Landing Page

A high-end, fully responsive landing page inspired by Discord, built to showcase advanced front-end development skills with a strong focus on modern UI/UX, performance, and visual polish.

## 🔥 Features

* ⚡ **Advanced Animations**

  * Scroll-based reveal (Intersection Observer)
  * Smooth transitions and motion design
  * Parallax effects based on cursor movement

* 🎯 **Interactive UI**

  * Magnetic buttons (hover physics effect)
  * Dynamic cursor glow
  * Interactive text reacting to mouse movement

* 🎨 **Modern Design**

  * Glassmorphism (blur + transparency)
  * Gradient systems and depth
  * Clean and professional layout inspired by top tech companies

* 📱 **Fully Responsive**

  * Optimized for desktop, tablet, and mobile
  * Adaptive layout using CSS Grid & Flexbox

* 🧠 **Performance & Architecture**

  * Optimized rendering with `requestAnimationFrame`
  * Lazy animations using `IntersectionObserver`
  * Clean and scalable structure

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**

  * Custom Properties (CSS Variables)
  * Flexbox & Grid
  * Backdrop Filters (Glass Effect)
